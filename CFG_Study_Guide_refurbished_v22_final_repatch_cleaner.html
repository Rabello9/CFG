Abrir no navegador para funcionar (iPhone)

Alguns visualizadores (ex.: preview do app/Quick Look) bloqueiam JavaScript. Este guia um dashboard interativo (React) e precisa do navegador completo.

  1. Toque no boto de Compartilhar (cone quadrado com seta).
  2. Escolha Abrir no Safari (ou Abrir no Navegador).
  3. Se estiver no app Arquivos: Compartilhar Abrir no Safari.

Se o dashboard carregar normalmente, esta tela some automaticamente.
